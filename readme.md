# 🚀 AI Resume Analyzer

An AI-based web application that analyzes resumes and compares them with job descriptions to calculate match scores and identify skill gaps.

## 🔥 Features

* Upload resume (PDF)
* Extract and analyze skills using NLP
* Match resume with job description
* Calculate match percentage
* Identify missing skills

## 🛠️ Tech Stack

* Python
* Streamlit
* NLP (TF-IDF, text processing)
* PDF parsing

## 🌐 Live App

👉 https://ai-resume-analyzer-rr.streamlit.app/

## 📌 How it Works

1. Upload your resume
2. Paste a job description
3. The system extracts skills and compares them
4. Get match score and missing skills
